# BOJ 11437 - LCA

## 문제 요약

N개의 노드로 이루어진 트리에서 두 노드의 최소 공통 조상(LCA) 쿼리 M개 처리.
- N ≤ 50,000 / M ≤ 10,000

## 접근법

**나이브 LCA**: 두 노드의 깊이를 맞추고 함께 올라가며 만나는 첫 공통 조상 찾기.
- 전처리 O(n), 쿼리당 O(n) → 전체 O(NM)
- N, M이 작으므로 통과 가능

더 큰 제약엔 → [BOJ 11438 LCA 2](../05-tree/02-boj-11438-lca2.md) (희소 테이블 사용)

## 복잡도

| | 시간 |
|-|------|
| 전처리 (BFS) | O(N) |
| 쿼리 1회 | O(N) |
| 전체 | O(NM) |

## 코드

```cpp
#include <cstdio>
#include <cstring>

const int MAXN = 50005;

// 인접 리스트 (양방향)
int head[MAXN], nxt[MAXN*2], to[MAXN*2], ecnt;
void addEdge(int u, int v) {
    to[++ecnt] = v; nxt[ecnt] = head[u]; head[u] = ecnt;
    to[++ecnt] = u; nxt[ecnt] = head[v]; head[v] = ecnt;
}

int depth[MAXN], parent[MAXN];
bool visited[MAXN];

// BFS로 depth, parent 초기화
void bfs(int root, int N) {
    int q[MAXN], front = 0, back = 0;
    memset(visited, false, sizeof(visited));
    visited[root] = true;
    depth[root] = 0; parent[root] = root;
    q[back++] = root;
    while (front < back) {
        int u = q[front++];
        for (int e = head[u]; e; e = nxt[e]) {
            int v = to[e];
            if (!visited[v]) {
                visited[v] = true;
                depth[v] = depth[u] + 1;
                parent[v] = u;
                q[back++] = v;
            }
        }
    }
}

int lca(int u, int v) {
    while (depth[u] > depth[v]) u = parent[u];
    while (depth[v] > depth[u]) v = parent[v];
    while (u != v) { u = parent[u]; v = parent[v]; }
    return u;
}

int main() {
    int N;
    scanf("%d", &N);
    for (int i = 0; i < N - 1; i++) {
        int u, v;
        scanf("%d %d", &u, &v);
        addEdge(u, v);
    }
    bfs(1, N);

    int M;
    scanf("%d", &M);
    while (M--) {
        int u, v;
        scanf("%d %d", &u, &v);
        printf("%d\n", lca(u, v));
    }
    return 0;
}
```

## 풀이 포인트

- 루트를 1로 고정, BFS로 각 노드의 depth와 parent를 구함.
- LCA 쿼리: 깊이가 같아질 때까지 한 쪽을 올리고, 이후 함께 올라감.
