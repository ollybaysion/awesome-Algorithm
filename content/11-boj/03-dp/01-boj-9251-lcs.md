# BOJ 9251 - LCS

## 문제 요약

두 문자열의 **최장 공통 부분 수열(LCS)** 길이 출력.
- 문자열 길이 ≤ 1000

## 접근법

2D DP:

```
dp[i][j] = A[1..i]와 B[1..j]의 LCS 길이
if A[i] == B[j]: dp[i][j] = dp[i-1][j-1] + 1
else:            dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

## 복잡도

O(n·m) 시간, O(n·m) 공간

## 코드

```cpp
#include <cstdio>
#include <cstring>

const int MAXN = 1005;
char A[MAXN], B[MAXN];
int  dp[MAXN][MAXN];

int max(int a, int b) { return a > b ? a : b; }

int main() {
    scanf("%s %s", A + 1, B + 1);
    int n = strlen(A + 1);
    int m = strlen(B + 1);

    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            if (A[i] == B[j])
                dp[i][j] = dp[i-1][j-1] + 1;
            else
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
        }
    }

    printf("%d\n", dp[n][m]);
    return 0;
}
```

## 풀이 포인트

- 1-indexed 문자열 사용으로 dp 점화식을 자연스럽게 표현.
- `dp[0][*]`와 `dp[*][0]`은 0으로 초기화 (배열 전역 선언).

## 변형 문제

### LCS 실제 수열 복원

```cpp
void printLCS(int i, int j) {
    if (i == 0 || j == 0) return;
    if (A[i] == B[j]) {
        printLCS(i-1, j-1);
        putchar(A[i]);
    } else if (dp[i-1][j] >= dp[i][j-1]) {
        printLCS(i-1, j);
    } else {
        printLCS(i, j-1);
    }
}
```

### 공간 최적화 (O(m) 공간)

```cpp
int prev[MAXN], cur[MAXN];
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        if (A[i] == B[j]) cur[j] = prev[j-1] + 1;
        else cur[j] = max(prev[j], cur[j-1]);
    }
    memcpy(prev, cur, sizeof(int) * (m+1));
}
```
