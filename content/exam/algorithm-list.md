# 알고리즘 목록

알고리즘을 카테고리별로 정리한 레퍼런스 목록입니다.
각 항목 옆의 복잡도는 일반적인 경우를 기준으로 합니다.

---

## 자료구조

| 알고리즘 / 구조 | 시간 복잡도 | 비고 |
|----------------|-------------|------|
| 배열 (Array) | 접근 O(1), 탐색 O(n) | 연속 메모리 |
| 연결 리스트 | 접근 O(n), 삽입 O(1) | 포인터 체인 |
| 스택 / 큐 | O(1) all ops | LIFO / FIFO |
| 힙 (이진 힙) | 삽입·삭제 O(log n) | 우선순위 큐 |
| 해시 테이블 | 평균 O(1) | 충돌 주의 |
| 세그먼트 트리 | 쿼리·업데이트 O(log n) | 구간 집계 |
| 펜윅 트리 (BIT) | 쿼리·업데이트 O(log n) | 구현 간단 |
| 유니온-파인드 | O(α(n)) ≈ O(1) | 분리 집합 |
| 트라이 (Trie) | 삽입·탐색 O(L) | 문자열 집합 |
| 스파스 테이블 | 전처리 O(n log n), 쿼리 O(1) | RMQ, 불변 배열 |

## 정렬

| 알고리즘 | 시간 복잡도 | 공간 | 비고 |
|----------|-------------|------|------|
| 버블 정렬 | O(n²) | O(1) | 교육용 |
| 선택 정렬 | O(n²) | O(1) | 교육용 |
| 삽입 정렬 | O(n²) / O(n)* | O(1) | *거의 정렬된 경우 |
| 병합 정렬 | O(n log n) | O(n) | 안정 정렬 |
| 퀵 정렬 | O(n log n) avg | O(log n) | 불안정, 캐시 효율 |
| 힙 정렬 | O(n log n) | O(1) | 불안정 |
| 계수 정렬 | O(n + k) | O(k) | 정수, 범위 제한 |
| 기수 정렬 | O(d·n) | O(n) | 정수, d = 자릿수 |

## 탐색

| 알고리즘 | 시간 복잡도 | 비고 |
|----------|-------------|------|
| 선형 탐색 | O(n) | |
| 이진 탐색 | O(log n) | 정렬된 배열 필요 |
| BFS | O(V + E) | 최단 경로 (가중치 동일) |
| DFS | O(V + E) | 경로 탐색, 사이클 |

---

## 그래프

| 알고리즘 | 시간 복잡도 | 적용 |
|----------|-------------|------|
| 다익스트라 | O((V+E) log V) | 단일 출발, 음수 없음 |
| 벨만-포드 | O(VE) | 음수 간선, 음수 사이클 검출 |
| 플로이드-워샬 | O(V³) | 전체 쌍 최단 경로 |
| 크루스칼 | O(E log E) | MST, 유니온-파인드 |
| 프림 | O(E log V) | MST, 힙 이용 |
| 위상 정렬 (Kahn) | O(V + E) | DAG 순서 |
| SCC (Tarjan / Kosaraju) | O(V + E) | 강연결 요소 |
| 2-SAT | O(V + E) | SCC 기반 |
| 최대 유량 (Dinic) | O(V²E) | 이분 매칭 포함 |
| LCA (희소 테이블) | 전처리 O(n log n), 쿼리 O(log n) | 트리 공통 조상 |
| HLD | O(n log²n) | 트리 경로 쿼리 |

## 동적 프로그래밍

| 유형 | 복잡도 | 예시 문제 |
|------|--------|-----------|
| 1D DP | O(n) ~ O(n²) | LIS, 냅색 |
| 2D DP | O(n²) ~ O(n³) | LCS, 구간 DP |
| 비트마스크 DP | O(2ⁿ · n) | TSP |
| 트리 DP | O(n) | 독립 집합, 루트 선택 |
| CHT 최적화 | O(n log n) | 기울기가 단조인 경우 |
| 분할 정복 최적화 | O(kn log n) | opt 단조 |
| Knuth 최적화 | O(n²) | 사각 부등식 + 구간 DP |

## 문자열

| 알고리즘 | 시간 복잡도 | 적용 |
|----------|-------------|------|
| KMP | O(n + m) | 단일 패턴 매칭 |
| Z-알고리즘 | O(n + m) | Z 배열, 매칭 |
| 라빈-카프 (해싱) | O(n + m) avg | 다중 패턴, 롤링 해시 |
| 아호-코라식 | O(N + M + K) | 다중 패턴 동시 탐색 |
| 접미사 배열 (SA) | O(n log n) | 접미사 정렬, LCP |
| Manacher | O(n) | 팰린드롬 반지름 |
| Eertree | O(n) | 고유 팰린드롬 수 |

## 수학

| 알고리즘 | 복잡도 | 비고 |
|----------|--------|------|
| 에라토스테네스의 체 | O(n log log n) | 소수 목록 |
| 유클리드 GCD | O(log min(a,b)) | 최대공약수 |
| 확장 유클리드 | O(log min(a,b)) | ax+by=gcd |
| 모듈러 거듭제곱 | O(log e) | 빠른 지수 |
| 모듈러 역원 | O(log p) | 페르마 소정리 (p 소수) |
| 중국인의 나머지 정리 | O(log n) | 연립 합동 |
| 뤼카 정리 | O(p²) 전처리 | 이항계수 mod p |
| 밀러-라빈 소수 판별 | O(k log²n) | 확률적 판별 |
| 폴라드 로 인수 분해 | O(n^1/4) | |
